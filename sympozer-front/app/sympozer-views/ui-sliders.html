Using sliders is as easy as adding a ui-bs-slider attribute and setting slider properties.

<input type="text"
   value=""
   ui-bs-slider
   data-slider-min="0"
   data-slider-max="100"
   data-slider-step="10"
   data-slider-value="[10, 90]"
   data-slider-selection="before"
   data-slider-tooltip="show"
   data-slider-handle="round">

jQuery Knob is a downward compatible, canvas based and touch friendly jQuery dial.

To use Knob, use Forza's knob element and set data-* properties like below. No JS code needed!

    <knob value="25" data-width="180" data-displayInput="false" data-thickness=".15"></knob>
