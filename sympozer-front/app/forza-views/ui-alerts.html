We've written a small adapter for Pines Notifications available as a service that allows you to use Pines Notification with Angular without breaking any rules. Just inject pinesNotification in your controller and you're ready to go!

//inside the controller
$scope.simpleInfo = function () {
  pinesNotifications.notify({
    title: 'New Thing',
    text: 'Just to let you know, something happened.',
    type: 'info'
  });
};

Pulsating Elements

jQuery.pulsate.js adds a pulsating effect to elements. Useful for drawing the users attention.

Use Pulsate by adding a data-pulsate attribute to the element and passing it the options object like so: data-pulsate="{glow:false}". Pretty neat, eh?
