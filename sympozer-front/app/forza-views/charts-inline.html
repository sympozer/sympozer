Sparkline Variations

Sparklines are very small, inline charts that can be drawn using data supplied either inline in the HTML, or via JavaScript. They can, however, be customized to fit even bigger divs.

Just add an sparklines attribute in your code and let Sparklines handle the rest!

Pie charts

Composite graphs are created using Forza's sparklines-composite directive like so:

<sparklines-composite>
    <span sparklines="{type: 'bar'}" values="5,4,7,6,9,5,8,2,6,4,6,7"></span>
    <span sparklines="{type: 'bar', lineColor: 'red'}" data-data="[1,33,3,9,2,7,5]"></span>
</sparklines-composite>
